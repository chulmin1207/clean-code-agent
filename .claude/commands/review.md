$ARGUMENTS 에 지정된 대상을 클린코드 기준으로 전면 리뷰하세요.

## 입력 판별

- 파일 경로 → 해당 파일 분석
- 디렉토리 경로 → 하위 소스 파일을 Glob으로 수집, 각각 분석 후 종합 리포트
- 인자 없음 → `git diff --name-only HEAD~1` 로 최근 커밋 변경 파일 자동 선택

## 실행 순서

1. 대상 파일 전체 읽기
2. 5가지 관점 분석 (가독성&명명 / 함수설계 / 중복 / 복잡도 / 주석)
3. 클린코드 점수 산출 (100점, 루브릭 적용)
4. 개선 우선순위 상위 3가지 제시

## 디렉토리 분석 시

- 파일별 개별 점수 + 전체 평균 점수 산출
- 프로젝트 전체에 걸친 공통 문제 패턴 식별
- 가장 시급한 파일 TOP 3 지목
